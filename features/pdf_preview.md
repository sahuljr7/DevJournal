# Feature: In-situ PDF Inspection

## Description
Integrated document viewer for frictionless technical research and verification within the documentation flow.

## How it Works
- Detects the `application/pdf` MIME type for consigned sources.
- Provides a dedicated "PDF Preview" toggle button.
- Expands the view to a large vertical canvas (600px) with "Fit-to-Width" logic for optimal readability within the app frame.

## Usage
1. Locate a PDF attachment in the ledger or editor.
2. Click "Preview PDF".
3. Use the in-line viewer to inspect the document without leaving the application context.
